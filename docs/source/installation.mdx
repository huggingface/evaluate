# Installation

Before you start, you will need to setup your environment and install the appropriate packages. ðŸ¤— Evaluate is tested on **Python 3.7+**.

## Virtual environment

You should install ðŸ¤— Evaluate in a [virtual environment](https://docs.python.org/3/library/venv.html) to keep everything neat and tidy.

1. Create and navigate to your project directory:

   ```bash
   mkdir ~/my-project
   cd ~/my-project
   ```

2. Start a virtual environment inside the directory:

   ```bash
   python -m venv .env
   ```

3. Activate and deactivate the virtual environment with the following commands:

   ```bash
   # Activate the virtual environment
   source .env/bin/activate
   
   # Deactivate the virtual environment
   source .env/bin/deactivate
   ```

Once you have created your virtual environment, you can install ðŸ¤— Evaluate in it.

## pip

The most straightforward way to install ðŸ¤— Evaluate is with pip:

```bash

```

Run the following command to check if ðŸ¤— Evaluate has been properly installed:

```bash

```

This should return:

```bash

```
